# Apache Derby (apache-derby)
Apache Derby is an open-source relational database implemented entirely in Java. It can be embedded in Java applications or used in a client/server configuration, providing a small footprint database engine with full SQL support and JDBC compliance.

### Apache Derby
Derby provides a standard JDBC API for database operations in both embedded and client/server modes, supporting SQL, stored procedures, triggers, and full transaction management.
